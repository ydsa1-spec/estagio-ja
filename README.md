# 1. Introdução

## 1.1 Propósito

Este documento define a visão geral do sistema EstágioJá, descrevendo os problemas que o projeto busca resolver, os usuários envolvidos, as funcionalidades principais e os requisitos do sistema.

O documento serve como base para organização e desenvolvimento do projeto durante o 4º bimestre.

---

## 1.2 Escopo

O EstágioJá é uma plataforma web voltada para estudantes que procuram oportunidades de estágio.

O sistema contempla:

- Cadastro de usuários
- Login e autenticação
- Busca de vagas de estágio
- Filtros por cidade e área
- Visualização de detalhes da vaga
- Salvamento de vagas favoritas
- Candidatura às vagas
- Perfil do usuário
- Acompanhamento de candidaturas

Fora do escopo da versão 1.0:

- Aplicativo mobile
- Integração com APIs externas
- Chat em tempo real
- Sistema premium

---

# 2. Posicionamento

## 2.1 Oportunidade de Negócio

Muitos estudantes possuem dificuldade em encontrar vagas de estágio organizadas e acessíveis.

Grande parte das oportunidades estão espalhadas em diferentes plataformas, dificultando a busca, candidatura e acompanhamento das vagas.

---

## 2.2 Declaração do Problema

O problema da dificuldade de acesso a vagas de estágio afeta estudantes que buscam oportunidades profissionais.

Isso gera dificuldade de inserção no mercado de trabalho, perda de oportunidades e falta de organização no processo de candidatura.

Uma solução eficiente seria uma plataforma centralizada para busca e candidatura de vagas.

---

## 2.3 Declaração de Posição do Produto

Para estudantes que buscam oportunidades de estágio,

o EstágioJá é uma plataforma web de busca e gerenciamento de vagas,

que oferece cadastro, pesquisa de vagas, filtros e candidatura online,

diferente de plataformas genéricas e desorganizadas,

nosso sistema busca facilitar o acesso dos estudantes ao mercado de trabalho.
